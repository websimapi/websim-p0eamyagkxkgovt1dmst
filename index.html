Quick Guide

Input Methods

Piano: Tap keys or type A-G to add notes.

Parts: Toggle Treble / Bass in the toolbar to switch staves.

Octave: Use [ and ] keys to shift (Range 0-8).

Chords: Toggle ⋮ to stack notes on input.

Rests: Toggle 𝄽 or press R.

Editing Tools

  • Selection: Click note. Shift+Click for range.
  • Copy/Paste: Buttons or Ctrl+C / Ctrl+V.
  • Transpose: ↑ / ↓ keys or arrow buttons.
  • Enharmonic: ♯/♭ swaps spelling (e.g. C# ↔ Db).
  • Ties/Slurs: Select note(s) → ‿ (Tie) or Slur icon.
  • Dynamics: Select range → < / > (Hairpins).
  • Articulations: . Staccato, > Accent, ^ Marcato.
  • Lyrics: Tap 'Lyrics' to add text to selected note.
